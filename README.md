#GXBSystem#

这是套基于ThinkPHP3.1.3的后台基础框架，提供系统配置、用户权限管理、新闻管理、文件管理与数据库备份等基础功能。旨在为衍生其他产品项目打好基石，减少后台开发的基本工时。
